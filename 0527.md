## Flutterとtextを出す
```
 body:
   new Text(
   "Flutter",
     style: new TextStyle(fontSize:64.0, 
     color: const Color(0xFF000000),      
     fontWeight: FontWeight.w200,      
     fontFamily: "Roboto"),               
 ),
```
| fontSize | フォントサイズ。double値で指定 |  
:----:|:----
| fontWeight | フォントの太さ |
| fontFamily | フォントファミリー。テキストで指定 |
| fontStyle | フォントスタイル |
| color | テキスト色 |
