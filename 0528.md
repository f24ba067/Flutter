# ボタン・ウィジェット
### TextButtonの基本形
```
TextButton(Key:null,
  onPressed: 関数,
  child: ウィジェット
)
```
### iconButtonの利用例
```
new IconButton(
  icon: const Icon(Icons.insert_emoticon), 
  onPressed:iconButtonPressed,
  iconSize: 48.0,
  color: const Color(0xFF000000),
),
```
| icon | 表示するアイコン | 
:----:|:----
| iconSize | アイコンサイズ |
| color | アイコンの色 |
| onPressed | クリックしたときに実行するメソッド |
